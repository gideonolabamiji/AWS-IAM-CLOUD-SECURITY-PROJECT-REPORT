# AWS-IAM-CLOUD-SECURITY-PROJECT-REPORT
The AWS IAM Cloud Security Project Report simply explains how user access and permissions were set up and managed in Amazon Web Services to keep the cloud environment secure and ensure only the right people can access the right resources.
This Project was completed in Amazon Web Services (AWS), focusing on Identity and Access Management (IAM). The goal was to create least privilege policy, attach it to a user group, and verify that the policy correctly restricts actions on two Amazon EC2 instances which are customer service and Audit
